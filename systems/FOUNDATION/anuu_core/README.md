# Anuu Core (Foundation)

This module contains the **Source Truth** of the system.

- **Unified JSON Schema:** Formatting memories and identity data.
- **Python Parsers:** Utilities to read/write the Anuu state.
- **Validation:** Ensuring memory integrity.

## Purpose
To provide a dependency-free library that any agent (local, cloud, mobile) can import to "understand" Anuu.
