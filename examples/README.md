# Anuu_Verse Examples

Executable examples to get you started quickly.

## Running Examples

```bash
cd examples
python basic_task.py
python agent_selection.py
```

## Examples Included

| File | Description |
|------|-------------|
| `basic_task.py` | Run a simple task with Anuu |
| `agent_selection.py` | Use specific agents (Kali, Kilonova) |
